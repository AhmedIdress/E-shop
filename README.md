# e_shop

A new Flutter project.

## Getting Started

an e-commerce app with flutter and convert the XD to dart code. App have splash screen, authentication screens ,home screen have bottom navigation bar have (home, categories, favorite, cart and profile) screens, side menu and checkout screen with payment methods and I have used provider as a state management , screenutlis as a responsive package and go on to connect with API    (app still in development mode)
